# 독해 훈련 (dokhae)

국어 비문학 지문을 Claude가 난이도별로 생성해 문제풀이·요약·속독으로 '정확하고 빠른 읽기'를 훈련하는 플러그인입니다.

## 설치

```
/plugin install dokhae@ben3329-plugins
```

## 사용

```
/dokhae:dokhae [quiz|summary|speed|stats] [난이도1-5]
```

`/dokhae:dokhae quiz 4`처럼 모드와 난이도를 직접 줘도 되고, "독해 연습하자"처럼 자연어로 불러도 됩니다. 모드가 불분명하면 먼저 모드를 물어봅니다. 난이도를 생략하면 기본 3(표준)입니다.

## 모드

- **quiz** — 비문학 문제풀이. 사실·추론·구조·어휘·비판 유형 문항을 풀고 채점·오답분석합니다.
- **summary** — 요약·핵심 추출. 요지를 직접 쓰면 핵심포착·정확성·간결성 3축으로 채점합니다.
- **speed** — 속독 + 이해도. `date`로 읽기 시간을 측정해 CPM(분당 글자수)을 내고 이해도 문제로 검증합니다.
- **stats** — 누적 기록 분석. 정답률·CPM 추이와 약점을 분석해 다음 훈련을 추천합니다.

## 기록

세션별 정답률·CPM·약점 태그를 `~/.claude/dokhae/records.jsonl`에 JSON Lines로 한 줄씩 저장합니다. 다음 훈련 시작 전 최근 기록을 읽어 성과가 낮은 영역과 약점 유형을 우선 반영해 출제합니다.

## 동작

- 지문은 Claude가 난이도/영역별로 직접 생성합니다.
- 정답·해설은 채점 전까지 노출하지 않습니다.
- 채점은 지문 내부 근거에 기반합니다.
- 명령(`/dokhae:dokhae`)이 진입점이고, 지문 생성·채점·기록 로직은 `dokhae-trainer` 스킬에 있습니다.
